# 요구사항 명세

1. 작업자의 생체, 상태 데이터를 분석하여 관리자의 화면에 정보 구현 및 알람
 - 작업자의 스마트 밴드로부터 가속도, 방위각, 좌표 등의 데이터를 받아서 행동분석
 - 인간의 기본 행동 특성을 분석하여 급격한 방향전환, 가속도 등을 분석하여 낙상, 추락, 활동량 분석

2. 데이터 분석
 - 작업자 센싱 데이터 분석시에 레이블 데이터는 없음
 - 상태 정상/비정상 작업자의 센싱 정보의 다른 패턴을 발견하는 비지도 학습 사용 => 군집분석 / 의사결정트리분석

3. 화면 설계
 - 지도창에 특정 건설 현장에 투입된 작업자들의 목록을 표출하고
   각 작업자를 아이콘으로 표시해 해당 아이콘을 클릭하면 작업자의 신상 정보를 보여주게 함
 - 특정 작업자의 작업 이력을 선택하면 작업자의 생체 신호, 자이로 센싱 데이터를 그래프로 표출
 - 낙상, 추락 등의 위험 작업자 위치를 지도창에 표출, 이 중 특정 작업자를 선택하면 해당 작업의 위험 데이터와 그래프를 표출
=> 웹 화면 상에 건설 현장의 지도(위험구역 표시), 지도상에 각 작업자 아이콘으로 표시되고 있음(위치에 따라 변동) + 투입 작업자들 목록
=> 아이콘을 클릭하면 작업자의 작업 이력 / 생체, 상태 데이터 / 맡은 작업의 위험 데이터 그래프 표출
=> 작업자가 지도상 위험구역에 들어가거나 실시간 데이터 상에 문제가 생기거나 안전에 위험 확률이 높아지면 관리자에 알림
